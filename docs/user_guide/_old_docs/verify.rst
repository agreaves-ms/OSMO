================================================
Verify Installation
================================================

.. |check| raw:: html

  <input type="checkbox">

|check| Validate the installation by running a sample workflow. Refer to :ref:`getting_started_ui`

|check| Validate the ``default`` dataset bucket. Refer to the instructions in :ref:`ds_upload` and run the first example on uploading to ``TestDataset``

|check| Validate a ``specific`` dataset bucket. Refer to the instructions in :ref:`ds_upload` and run the second example on uploading to bucket ``TestBucket`` and replace ``TestBucket`` with the desired bucket

.. note::
  Refer to :ref:`credentials_data` section to resolve any credentials issue you might encounter
